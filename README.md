# move-sui
sui by move. All IN MOVE

## 常用命令
要检查Sui是否已安装
```
which sui
```

将Sui客户端连接到网络
```
sui client
```

要检查当前可用的环境别名
```
sui client envs
```

切换活动网络
```
sui client switch --env <ALIAS>
```

查看当前钱包地址
```
sui client active-address
```

查看当前地址中的币
```
sui client gas
```
## 开发命令
创建新项目
```
sui move new project_name
```
